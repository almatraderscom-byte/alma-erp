# Alma ERP Android APK

Place the built APK here for self-hosted download:

- File name: `alma-erp.apk`
- Download page: `/download.html` (also `/app/download` redirects here)
- Build: `npm run mobile:apk:release` → copy from `mobile/dist/`

Or host on Google Drive and set `NEXT_PUBLIC_ANDROID_APK_URL` in Vercel.
