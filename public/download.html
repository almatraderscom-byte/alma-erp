ইনস্টল করার নিয়ম

  1. উপরের বাটনে ট্যাপ করে APK ডাউনলোড করুন
  2. ডাউনলোড শেষে ফাইলে ট্যাপ করুন → Install
  3. “Unknown sources” চাইলে Settings থেকে browser-এর জন্য allow করুন
  4. ইনস্টল হয়ে গেলে Home screen-এ Alma ERP icon খুলে login করুন
  5. Login-এর পর Allow notifications চাপুন — lock-screen alert চালু হবে
  6. সাধারণ আপডেটে APK আবার ইনস্টল করতে হবে না — অ্যাপে রিফ্রেশ করলেই নতুন ভার্সন চলবে

লোড হচ্ছে কিন্তু বাটন আসে না? এই page সরাসরি খুলুন — কোনো login লাগে না।
