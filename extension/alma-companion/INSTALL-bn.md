# ALMA Companion — Chrome-এ বসানোর নিয়ম (Sir-এর জন্য)

এই এক্সটেনশনটা বসালে আপনার এজেন্ট **আপনার নিজের Chrome-এ, আপনার লগইন দিয়ে** কাজ করতে
পারবে — যেমন আপনার Facebook পেজ খুলে দেখা, কোনো সাইট রিভিউ করা — আর আপনি পুরোটা **লাইভ
দেখবেন**। কোথাও আপনার পাসওয়ার্ড লাগবে না; আপনি আগে থেকে যে সাইটে লগইন করা আছেন, সেই
লগইনই কাজে লাগবে।

## ১) এক্সটেনশনটা বসান (একবারই করতে হবে)

1. Chrome খুলে অ্যাড্রেস বারে লিখুন: `chrome://extensions`
2. ডান দিকে উপরে **Developer mode** চালু করুন।
3. **Load unpacked** বাটনে ক্লিক করুন।
4. এই ফোল্ডারটা বেছে দিন: `extension/alma-companion`
5. টুলবারে একটা সোনালি **ALMA Companion** আইকন দেখা যাবে।

## ২) যুক্ত করুন (pair)

1. চ্যাটে এজেন্টকে বলুন: **"live browser চালু করো"** তারপর **"pair code দাও"**।
2. এজেন্ট একটা **এক-বারের কোড** দেবে (যেমন `4F9K-2T7Q`), প্রায় ১০ মিনিট চলবে।
3. টুলবারের ALMA Companion আইকনে ক্লিক করুন → কোডটা বসান → **যুক্ত করুন** চাপুন।
4. আইকনের পাশে সবুজ ● এলে বুঝবেন যুক্ত হয়েছে।

> **নাম দেওয়ার নিয়ম (গুরুত্বপূর্ণ):** পপআপে একটা **নাম** ঘর আছে। এটা নিজে থেকেই
> আপনার কম্পিউটার বুঝে নাম বসিয়ে দেয় — Mac-এ **"My Mac Chrome"**, Windows-এ
> **"My Windows Chrome"**। **এই নামটা বদলাবেন না** (বা আলাদা রাখলেও Mac ও Windows-এর
> নাম যেন আলাদা হয়)। কারণ আপনি যদি একসাথে Mac ও Windows দুটোতেই যুক্ত করেন, তখন
> এজেন্টকে বলবেন *"Windows Chrome-এ কাজ করো"* — নাম আলাদা থাকলেই সে ঠিক ব্রাউজারটা বেছে নিতে পারবে।

## ২ক) Windows-এর Chrome-এ বসানো (একই এক্সটেনশন)

এই এক্সটেনশন Mac আর Windows — দুটোতেই হুবহু একইভাবে চলে, কোনো আলাদা ভার্সন লাগে না।

1. `alma-companion` ফোল্ডারটা আপনার Windows PC-তে নিয়ে যান (pen-drive, Google Drive,
   বা zip ফাইল দিয়ে — zip হলে আগে **Extract** করে খুলে নিন)।
2. Windows-এর Chrome খুলে অ্যাড্রেস বারে: `chrome://extensions`
3. ডানে উপরে **Developer mode** চালু করুন → **Load unpacked** → ওই `alma-companion` ফোল্ডারটা বেছে দিন।
4. এরপর উপরের **"২) যুক্ত করুন"** ধাপ হুবহু অনুসরণ করুন। নাম ঘরে নিজে থেকেই
   **"My Windows Chrome"** বসবে — সেটাই রাখুন।
5. এখন Mac ও Windows — দুটোই একসাথে অনলাইন থাকতে পারবে। এজেন্টকে বললে সে জিজ্ঞেস করবে
   *"কোনটাতে কাজ করব?"*, নয়তো আপনি সরাসরি বলবেন *"Mac Chrome-এ যাও"* বা *"Windows Chrome-এ যাও"*।

## ৩) ব্যবহার

- এজেন্টকে বলুন, যেমন: *"আমার Facebook পেজে যাও, কোথাও কোনো ভুল আছে কিনা দেখো।"*
- এজেন্ট আপনার ওই ট্যাবেই কাজ করবে; আপনি সব নিজের চোখে দেখবেন।
- প্রতিটি ধাপের একটা স্ক্রিনশটও এজেন্ট আপনাকে দেখাবে।

## নিরাপত্তা (গুরুত্বপূর্ণ)

- **থামান বাটন:** এক্সটেনশন পপআপে যেকোনো সময় **"থামান"** চাপলে এজেন্ট ওই মুহূর্তেই সব
  কাজ বন্ধ করে দেবে — সার্ভার যা-ই বলুক।
- এজেন্ট কখনো **শেষ "Send / Post / Pay / Buy / Confirm / Delete"** নিজে চাপবে না —
  ফর্ম ভরে দেবে, পেজে নিয়ে যাবে, কিন্তু ওই শেষ ক্লিকটা আপনি নিজে করবেন।
- পাসওয়ার্ড, ব্যাংক/কার্ডের তথ্য এজেন্ট কখনো টাইপ করবে না।
- কাজ শেষে চাইলে পপআপে **"এই ব্রাউজার খুলে ফেলুন (unpair)"** চেপে যোগ বিচ্ছিন্ন করতে পারবেন।

## কখন কাজ করবে না

- Chrome বন্ধ থাকলে, বা পপআপে **থামানো** থাকলে।
- লাইভ ব্রাউজার **বন্ধ** (`live browser বন্ধ করো`) থাকলে।
- এই দুটোর যেকোনো একটাই আপনার পূর্ণ নিয়ন্ত্রণ — তাই নিশ্চিন্ত থাকুন, Sir।
