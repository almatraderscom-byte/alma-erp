# ALMA ERP iOS — ফাইনাল ডিভাইস টেস্ট চেকলিস্ট (Boss-এর জন্য)

**কখন:** IOSP-1 থেকে IOSP-9 সব একসাথে যাওয়া **একটাই TestFlight বিল্ড** ইনস্টল করার পর।
**কীভাবে:** প্রতিটা লাইন নিজের iPhone-এ করে দেখুন — ঠিক থাকলে ✓, সমস্যা হলে ✗ দিয়ে এক লাইনে কী হয়েছে লিখে পাঠান। সব মিলিয়ে ~১৫ মিনিট।

## ১. চালু হওয়া + লগইন (২ মিনিট)
- [ ] অ্যাপ খুললে Face ID দিয়ে ঢোকা যায়, ক্র্যাশ নেই
- [ ] Dashboard-এ আজকের আসল টাকার অঙ্ক দেখা যায় (৳ সংখ্যা, অর্ডার)
- [ ] অ্যাপ বন্ধ করে আবার খুললে আবার লগইন চায় না

## ২. পাঁচটা মূল ট্যাব (৩ মিনিট)
- [ ] Dashboard / Orders / Assistant / Approvals / More — পাঁচটাই খোলে, কোনোটা সাদা/ফাঁকা না
- [ ] Orders-এ আসল অর্ডার লিস্ট, টান দিলে মসৃণ scroll
- [ ] Approvals-এ pending কার্ডে Approve/Reject বাটন কাজ করে (একটা টেস্ট আইটেম approve করে দেখুন)
- [ ] ট্যাব বদলালে হালকা কাঁপুনি (haptic) লাগে

## ৩. টাকার স্ক্রিন (৩ মিনিট)
- [ ] More → Payroll: বেতন বাজেট/liability সংখ্যা আসে, ৩টা ব্যবসার চিপ (Alma/CDIT/Trading) বদলালে সংখ্যা বদলায়
- [ ] More → Office Fund আর Expenses: সংখ্যা ওয়েবের সাথে মেলে
- [ ] Trading accounts: ব্যালেন্স ৳ সংখ্যা আসে

## ৪. এজেন্ট (Assistant) (৩ মিনিট)
- [ ] বাংলায় একটা প্রশ্ন পাঠান ("আজকের sales কত?") — উত্তর আসে
- [ ] ভয়েস বাটনে ধরে কথা বলুন — লেখা হয়ে যায়, উত্তরের আওয়াজ আসে
- [ ] চ্যাটে ছবি attach করা যায়

## ৫. নোটিফিকেশন + কল (২ মিনিট — শুধু আসল ফোনেই হয়)
- [ ] কাউকে দিয়ে একটা approval পাঠান — ফোনে push নোটিফিকেশন আসে
- [ ] নোটিফিকেশনে চাপ দিলে সোজা ঠিক পেজে যায় (লগইন পেজে ফেলে না)
- [ ] অফিস intercom কল এলে রিং হয়, ধরা যায়, আওয়াজ পরিষ্কার

## ৬. চোখের আরাম (Accessibility — ঐচ্ছিক, ২ মিনিট)
- [ ] iPhone Settings → Display → Text Size সর্বোচ্চ করে অ্যাপ খুলুন — টাকার সংখ্যা কাটা যায় না, বোতাম চাপা যায়
- [ ] Settings → Accessibility → Motion → Reduce Motion চালু করলে অ্যাপে দপদপানি/নাচানাচি বন্ধ থাকে
- [ ] টেস্ট শেষে দুটোই আগের মতো ফিরিয়ে দিন

## ৭. সবশেষে
- [ ] ১০ মিনিট ব্যবহারে ফোন গরম হয়নি, ব্যাটারি হঠাৎ কমেনি
- [ ] কোথাও দুইটা হেডার / দুইটা নিচের বার দেখা যায়নি

**কিছু ✗ হলে:** কোন লাইন + এক লাইনে কী দেখেছেন — এটুকুই যথেষ্ট, বাকিটা আমি বের করব।
