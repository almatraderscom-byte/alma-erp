# AIOS Request-Path Map (G01 / SPEC-001)

Frozen target request path (from `.aios/GLOBAL_AGENT_CONTRACT.md`):

```
Request
 -> Admission Control Plane      (G02)
 -> Cost Governor                (G04, arithmetic pre-authorises every model call)
 -> Context Compiler             (G05)
 -> Capability Broker            (G09)
 -> Policy/Approval              (G11, G12 — fail closed)
 -> Durable Workflow             (G14)
 -> Secure Tool Gateway          (G13 — every external side effect)
 -> Evidence Verification        (G19)
 -> Response Gate                 (bounded views to models)
 -> Audit + Cost + Evaluation    (G20)
```

G01 does not build these runtime stages. It freezes the **contract** every stage
must speak (`src/agent/contracts/component.ts`) and the **inventory** of the
current codebase so later groups migrate real call-sites rather than assumed
ones.

## How later groups use this

- **G02/G04**: every `provider call-site` above must be pre-authorised by the
  Cost Governor. The inventory is the checklist of sites to migrate.
- **G13**: every external side-effect site routes through the Secure Tool
  Gateway. The `telegram / facebook-graph / twilio` sites are the seed list.
- **Forbidden-dependency scan (SPEC-002)**: enforces the one-way rule — ERP code
  (`src/app`, `src/lib`) must never import `src/agent/**`.
